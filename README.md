# SQL 기반 앱 분석 지표 프로젝트

## 프로젝트 개요

이벤트 로그 데이터를 활용하여 음식 배달 앱의 핵심 비즈니스 지표를 SQL로 정의하고 분석했습니다.

- **데이터**: Firebase/GA4 스타일 이벤트 로그
- **기술**: BigQuery SQL

## 주요 발견사항

### 1. 낮은 전환율과 핵심 이탈 지점

**전체 전환율: 21.7%**

- 총 52,823명 중 11,467명만 결제 완료
- 업계 평균 대비 낮은 수치

**3대 이탈 지점:**

| 순위 | 이탈 구간               | 이탈률 | 이탈 인원 | 누적 영향 |
| ---- | ----------------------- | ------ | --------- | --------- |
| 1위  | 장바구니 → 결제         | 41%    | 5,765명   | 높음      |
| 2위  | 음식점 선택 → 메뉴 선택 | 39%    | 10,715명  | 매우 높음 |
| 3위  | 로그인 후 메인 화면     | 35%    | 17,145명  | 매우 높음 |

### 2. 리텐션 분석

**D1 Retention: 1-2%** <br/>
**Rolling D7 Retention: 8-10%**

**시기별 변화:**

- 2022년 8월: 3-5% (초기)
- 2022년 10월: 10-13% (피크)
- 2023년 1월: 5-10% (하락)

**해석:**

- 신규 유입은 있으나 재방문율 낮음
- 습관 형성 실패

### 3. 사용자 행동 패턴

**피크 타임: 9-13시 (오전 근무 시간)**

- 전체 트래픽의 40% 이상 집중
- 11-12시가 최고점 (각 8% 이상)
- 점심 주문 패턴 명확

**주요 이벤트 분포:**

| 이벤트              | 발생 횟수 | 사용자 수 | 사용률 |
| ------------------- | --------- | --------- | ------ |
| screen_view         | 400,122   | 52,823    | 100%   |
| click_login         | 105,107   | 49,678    | 94%    |
| click_food_category | 46,897    | 32,533    | 62%    |
| click_payment       | 13,159    | 11,467    | 22%    |

### 4. 로그인 행동 분석

**로그인 성공률: 100%**

- 로그인 시도한 사용자는 모두 성공
- 로그인 실패/오류 없음

**재로그인 빈도:**

- 1회만 로그인: 37%
- 2회 이상 로그인: 63%
- 평균 2.2회 로그인

**문제점:**

- 세션 유지 불충분 (짧은 타임아웃 추정)
- 로그인 후 메인 화면에서 명확한 다음 단계 부재

## 핵심 문제 진단

### 문제 1: 로그인 후 메인 화면 이탈 (35%, 17,145명)

**현상:**

```
로그인 완료 → screen_view 복귀 → 이탈
```

**원인:**

- 로그인 후 명확한 Call-to-Action 부재
- 메인 화면이 공백 상태
- 다음 단계 안내 없음

**개선안:**

- 로그인 직후 "오늘 뭐 드실래요?" 모달 표시
- 인기 메뉴 Top 3 자동 노출
- 개인화된 추천 배너

### 문제 2: 음식점 선택 이탈 (39%, 10,715명)

**현상:**

```
음식점 리스트 조회 → 메뉴 선택 안 함 → 이탈
```

**원인 (추정):**

- 음식점 리스트 매력도 부족
- 사진/리뷰 품질 낮음
- 필터링/정렬 옵션 부족

**개선안:**

- 음식점 사진 품질 개선
- 리뷰/별점 강조 표시
- 거리/배달시간 필터 추가
- 인기 음식점 우선 노출

### 문제 3: 장바구니 이탈 (41%, 5,765명)

**현상:**

```
장바구니 추가 → 결제 안 함 → 이탈
```

**원인 (추정):**

- 배달비/최소주문금액 확인 후 포기
- 결제 과정 복잡
- 추가 쿠폰/할인 부족

**개선안:**

- 배달비 사전 안내 (메뉴 선택 단계)
- 간편결제 추가
- 최소주문금액 미달 시 추천 메뉴 제시
- 장바구니 이탈자 리타겟팅 푸시 알림
